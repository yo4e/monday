# monday

GitHub作業用プロジェクトのサンドボックスです。

このリポジトリでは、月野テンプレクス（Monday）がGitHub連携でどこまで作業できるかを検証します。

## 現在の内容

- `index.html`: 静的HTMLのサンプルページ
- `style.css`: サンプルページ用のスタイル

## 運用メモ

- 初期化以外では、原則としてmainへ直接コミットしません。
- 変更はブランチを切り、Pull Requestで確認します。
- 小さなサンプルサイトやREADME更新などから試します。

## 次に試せること

- AstroやViteなどの軽量フレームワークへ移行する
